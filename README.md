# README

tools for validate something

## install

```shell
# npm
npm install @fewbadboy/validator -reg=https://mirrors.cloud.tencent.com/npm/

# yarn
yarn add @fewbadboy/validator --registry=https://mirrors.cloud.tencent.com/npm/
```
